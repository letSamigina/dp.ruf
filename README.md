# Roll und Flip
ダイスやコイントスを行う機能を追加するデータパックです。<bt>
※コイントスは未実装

## 【データパック名】
Roll und Flip Ver.1.0.0

### 《対応バージョン》
 - Minecraft Java Edition Ver.1.21.11
 - 他のデータパック、MODと併用する際の挙動は保証しません。

## 【主な機能】
 - ダイスを振る
   - 最大10個のダイスを1回で振れます
   - 値は1～100の間で操作可能です
     - 最小値は1で固定
   - 自分だけが確認できる「秘匿」と全体にロールの結果を公開する「公開」があります

順次機能を追加する予定です。

## 【操作方法】
 - 適切に導入が完了している場合、ワールドに入ったタイミングなどリロードがかかるタイミングでメッセージが表示されます。<br>
そこに記載されているキー（割り当てを変更していない場合はGキーです）を押すことで、当データパック専用のメニューを開くことができます。
